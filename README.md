# 📚 Adventure Digital Library Pro

## Features
- 📖 3-Language Reading (Arabic, Malayalam, English)
- 🎙️ TTS with Edge Natural Voices
- 🎬 Cinematic Visual Mode
- 📱 Offline-first with IndexedDB
- 🔄 Supabase Sync

## Setup
1. Clone repository
2. Add Supabase keys to Vercel
3. Deploy

## Technologies
- HTML5 + CSS3 + Vanilla JS
- Supabase (PostgreSQL)
- Vercel (Hosting)
- Web Speech API + Web Audio API
